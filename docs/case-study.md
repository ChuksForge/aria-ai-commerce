# Case Study — ARIA

## The Problem

Fashion e-commerce platforms struggle with product discovery.

Customers typically know:

* the occasion
* the budget
* the desired look

But they do not know:

* exact product names
* correct search terms

This creates three core gaps:

### 1. Vocabulary Gap

Users describe intent differently than systems expect

### 2. Context Gap

Preferences are not retained across interactions

### 3. Coordination Gap

Products are presented individually, not as complete solutions

---

## The Solution

ARIA introduces an intelligent layer that aligns with how users actually shop.

* Conversational interface captures intent naturally
* Semantic retrieval maps intent to relevant products
* Outfit generation connects individual items into complete looks

This transforms browsing into guided decision-making.

---

## Technical Approach

ARIA combines:

* semantic retrieval for efficient candidate selection
* language models for reasoning and generation
* structured outputs for system reliability

This hybrid approach balances:

* performance
* cost
* output quality

---

## Evaluation Approach

The system is evaluated across all major components:

* conversational understanding
* retrieval relevance
* outfit composition quality
* review summarisation accuracy

Both quantitative metrics and structured evaluation methods are used to ensure consistent performance.

---

## Business Impact

ARIA is designed to influence key e-commerce metrics:

* **Conversion rate** — improved discovery and relevance
* **Average order value** — multi-item recommendations
* **Return rate** — better sizing and expectation alignment

---

## Current Scope

* Controlled product catalogue
* Session-based context
* Single-language support

---

## Path to Production

* Integration with live product data
* Persistent user profiles
* Larger catalogue scaling
* Multi-language capability

---

## Summary

ARIA demonstrates how AI can be applied to solve real problems in e-commerce:

* understanding user intent
* improving product discovery
* guiding purchase decisions

It represents a shift from static interfaces to intelligent systems.
